# 期刊风格适配

## 🎯 功能简介

不同学术期刊有不同的写作风格偏好。本模块提供主流期刊的风格适配功能，帮助用户将论文调整为符合目标期刊要求的风格。

---

## 📚 支持的期刊风格

### 1. Nature 风格
**特点**：
- 简练有力，突出创新点
- 开篇直接点出研究意义
- 主动语态为主，语言富有张力
- 图表清晰，数据说话
- 字数限制：通常 3000-5000 字

**适用场景**：突破性研究、重大发现

---

### 2. Science 风格
**特点**：
- 结构极其清晰
- 逻辑严密，论证充分
- 方法和结果详尽
- 可复现性强
- 字数限制：通常 4000-6000 字

**适用场景**：方法创新、系统研究

---

### 3. IEEE 风格
**特点**：
- 技术细节丰富
- 公式和推导完整
- 算法描述清晰
- 实验验证充分
- 图表数量适中但信息密集

**适用场景**：工程、计算机、电子通信

---

### 4. Springer 风格
**特点**：
- 标准学术表达
- 结构完整规范
- 引用格式严谨
- 国际化程度高
- 字数限制：一般 6000-8000 字

**适用场景**：大多数学科的标准期刊

---

### 5. Cell / NEJM 风格
**特点**：
- 简洁精炼
- 临床导向（NEJM）
- 数据驱动
- 图表精美
- 辅助材料在线提供

**适用场景**：生命科学、医学

## 📋 Prompt 模板

### Prompt 2.1 - 适配 Nature 风格

```
请将我的论文适配为 **Nature** 期刊的写作风格。

【论文内容】
[粘贴你的论文]

【Nature 风格要求】

**1. 标题和摘要**
- 标题：简练有力（<15词），突出核心创新
- 摘要：200-250 词，清晰说明研究问题和主要发现
- 避免套话，直接陈述

**2. 引言**
- 开篇第一句就要抓住读者，点出研究意义
- 3-5 段，不超过 300 词
- 从研究空白直接切入，不要冗长背景
- 最后一句明确说明本文贡献

**3. 结果（Results）**
- 按逻辑顺序展示结果
- 每个要点配图或表
- 文字描述简洁，让图表说话
- 关键数字要突出

**4. 讨论**
- 不重复结果
- 深入解释发现的意义
- 与文献对话
- 展望未来方向
- 2-3 段即可

**5. 整体风格**
- 主动语态为主
- 短句为主，清晰有力
- 避免冗长修饰
- 每段一个核心观点
- 图表放在引用它的段落之后

【输出要求】
1. 说明你做了哪些主要调整
2. 给出适配后的完整论文
3. 检查字数是否符合 Nature 要求
4. 标注还需要人工调整的地方
```

### Prompt 2.2 - 适配 Science 风格

```
请将我的论文适配为 **Science** 期刊的写作风格。

【论文内容】
[粘贴你的论文]

【Science 风格要求】

**1. 结构要求**
- 标题：简洁，突出创新
- 摘要：150 词，涵盖研究问题、方法、主要发现
- 引言：研究背景 + 空白 + 本文贡献（简明）
- 方法：详细但不过分冗长
- 结果：系统展示，逻辑清晰
- 讨论：深入解释，展望未来

**2. 内容要求**
- 方法和结果要详尽，确保可复现
- 关键实验步骤不能省略
- 对照实验要说明
- 统计分析要规范

**3. 图表要求**
- 图表数量适中（通常 3-5 个）
- 每个图表信息密集
- 图注完整清晰
- 图例和标注规范

**4. 语言风格**
- 逻辑严密，论证充分
- 避免过度修饰
- 技术术语准确
- 公式推导清晰（如果有）

【输出要求】
1. 列出你做的主要调整
2. 给出适配后的论文
3. 检查可复现性是否充分
4. 标注需要补充的细节
```

### Prompt 2.3 - 适配 IEEE 风格

```
请将我的论文适配为 **IEEE** 期刊的写作风格。

【论文内容】
[粘贴你的论文]

【IEEE 风格要求】

**1. 结构标准**
- 标题：准确描述研究内容
- 摘要：200-250 词，标准结构（问题-方法-结果-结论）
- 关键词：3-5 个
- 引言：背景、相关工作、本文贡献、文章结构
- 系统模型/方法：详细描述
- 实验设计：说明数据集、对比方法、评估指标
- 结果与分析：充分展示，表格为主
- 讨论：分析结果、局限性
- 结论：总结 + 未来工作
- 参考文献：IEEE 格式

**2. 技术细节**
- 算法描述清晰（伪代码或详细步骤）
- 公式完整，编号规范
- 数学符号前后定义
- 实验设置详细（参数、硬件、软件）

**3. 图表规范**
- 表格为主，图表为辅
- 表格标题在上方
- 图标题在下方
- 所有图表都要在正文中引用
- IEEE 图例格式

**4. 引用格式**
- 文内引用：[1], [2,3], [4]-[6]
- 参考文献按出现顺序编号

**5. 语言风格**
- 被动语态为主（方法、结果部分）
- 准确的技术语言
- 避免模糊表述
- 客观陈述

【输出要求】
1. 列出调整清单
2. 给出适配后的论文
3. 检查参考文献格式
4. 标注需要补充的公式/算法
```

### Prompt 2.4 - 适配 Springer 风格

```
请将我的论文适配为 **Springer** 期刊的标准学术风格。

【论文内容】
[粘贴你的论文]

【Springer 风格要求】

**1. 结构完整**
- 标题：清晰描述研究内容
- 摘要：200-250 词
- 关键词：3-5 个
- 引言：充分但不冗长
- 文献综述：系统梳理
- 方法：详细描述
- 结果：清晰展示
- 讨论：深入分析
- 结论：总结
- 致谢（可选）
- 参考文献：Springer 格式

**2. 内容平衡**
- 引言和文献综述占全文 20-30%
- 方法和结果占 40-50%
- 讨论和结论占 20-30%

**3. 语言规范**
- 标准学术英语
- 避免口语化表达
- 术语使用一致
- 逻辑清晰

**4. 引用格式**
- Author-Date 格式或数字格式（根据具体期刊）
- 参考文献按字母顺序排列

【输出要求】
1. 列出主要调整
2. 给出适配后的论文
3. 检查各部分比例是否合适
4. 标注需要人工审阅的地方
```

### Prompt 2.5 - 自动检测并适配期刊风格

```
请分析我的论文，并推荐最适合的期刊风格，然后进行适配。

【论文内容】
[粘贴论文]

【分析任务】
1. 识别论文所属学科领域
2. 分析论文内容特点
3. 评估论文的创新程度
4. 推荐适合的期刊风格

【推荐逻辑】
- **Nature/Science**：突破性研究、重大发现、高影响力
- **IEEE**：工程、计算机、方法创新、技术细节丰富
- **Cell/NEJM**：生命科学、医学、临床研究
- **Springer**：标准学术研究、系统性研究

【适配要求】
根据推荐的期刊风格，完整适配论文：
1. 调整结构
2. 调整语言风格
3. 调整图表位置
4. 调整引用格式
5. 控制字数

【输出格式】
【风格推荐】
- 推荐期刊风格：[Nature/Science/IEEE/Springer/等]
- 推荐理由：[说明为什么推荐]
- 适配建议：[主要调整方向]

【适配后的论文】
[完整论文]

【适配说明】
- 主要调整：[列出]
- 需要人工检查：[列出]
```

### Prompt 2.6 - 跨期刊风格转换

```
请将我的论文从一种期刊风格转换为另一种。

【论文内容】
[粘贴论文]

【风格转换】
从：[原期刊风格，如 IEEE]
到：[目标期刊风格，如 Nature]

【转换策略】
1. 识别原风格的特征
2. 识别目标风格的特征
3. 逐项转换：
   - 结构调整
   - 语言风格调整
   - 图表调整
   - 引用格式转换
4. 保持核心内容和论点不变

【特别注意】
- 如果从详细风格（IEEE）转为简练风格（Nature）
  → 精简内容，突出重点，压缩图表
- 如果从简练风格（Nature）转为详细风格（IEEE）
  → 补充技术细节，扩展方法描述

【输出要求】
1. 列出风格差异对比
2. 列出转换操作清单
3. 给出转换后的论文
4. 标注可能丢失的信息（精简时）
5. 标注可能需要补充的细节（扩展时）
```

## 💡 使用建议

### 选择指南

- **不确定目标期刊** → Prompt 2.5（自动检测推荐）
- **目标是 Nature** → Prompt 2.1
- **目标是 Science** → Prompt 2.2
- **目标是 IEEE** → Prompt 2.3
- **目标是 Springer** → Prompt 2.4
- **已有论文，想换期刊** → Prompt 2.6

### 使用顺序

1. **先去AI化**（使用去AI化模块）
2. **再适配期刊**（使用本模块）
3. **最后一致性检查**（使用文风一致性检查）

### 重要提示

- 期刊适配会显著改变论文结构和风格
- 适配前备份原论文
- 适配后务必人工审阅
- 不同期刊对字数、图表数量要求不同
- 引文格式要仔细检查
