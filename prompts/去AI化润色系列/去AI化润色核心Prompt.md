# 去AI化润色核心 Prompt

## 🎯 模块简介

这是一个专门针对 AI 生成学术文本的润色系统，核心目标是：
1. 识别并去除模板化套话
2. 转换为自然、专业的真人学术语气
3. 保持学术严谨性的同时提升可读性
4. 支持多种期刊风格适配

---

## 📋 Prompt 模板

### Prompt 1.1 - 全文去AI化润色（核心推荐）

**使用场景**：对 AI 生成的学术文本进行全面的去AI化处理

```
我有一篇由 AI 辅助生成的学术论文，需要你帮我进行【去AI化润色】。

【原文内容】
[粘贴你的论文全文或段落]

【润色要求】
请按照以下原则进行润色：

【第一原则：去除模板化套话】
识别并删除或改写以下类型的套话：
- 开场白式："在当今时代"、"随着科技的发展"等
- 过渡句式："值得注意的是"、"值得一提的是"、"另一方面"等
- 模板句式："研究表明"、"根据数据"等（如果明显模板化）
- 重复强调："重要"、"关键"、"核心"等词汇过度使用
- 空洞表达："某种程度"、"某种程度上"等模糊表述

【第二原则：转换为真人学术语气】
将生硬的 AI 表达改写为学者常用的自然表达：
- ❌ 避免："因此，我们得出结论"
- ✅ 改为："综上，" 或 "结果表明"

- ❌ 避免："本文的研究表明，......"
- ✅ 改为："研究显示，..." 或直接陈述发现

- ❌ 避免："通过深入分析，我们发现......"
- ✅ 改为："分析表明，..." 或 "......"

- ❌ 避免："这表明......具有积极的意义"
- ✅ 改为："......有助于......" 或具体说明价值

【第三原则：保持学术严谨性】
- 不改变原意和核心论点
- 保留必要的学术术语和专业表达
- 保持逻辑结构的清晰性
- 论证要有理有据

【第四原则：提升可读性】
- 句式多样化（避免都是长句或都是短句）
- 段落结构合理（一个段落一个核心观点）
- 删除冗余表达（一句话说清楚的不要两句）
- 使用主动语态（适当使用，避免全被动）

【第五原则：统一文风】
- 全文使用一致的学术表达风格
- 时态统一（引言用现在时，方法用过去时）
- 术语使用一致（全文同一个概念用同一个术语）
- 引用格式统一

【输出要求】
请按以下格式输出：

【修改摘要】
- 识别出的套话类型：[列出发现的问题类型]
- 修改段落数：[X 段]
- 主要改进点：[列出 3-5 个关键改进]

【润色后的文本】
[粘贴完整润色后的文本]

【修改对比（可选）】
如果某处修改较大，可以简要说明：
- 原文：......
- 改为：......
- 原因：......

【质量检查】
最后请自评：
- AI 套话去除率：[估计百分比]
- 学术语气自然度：[1-10 分]
- 可读性提升：[低/中/高]
- 原意保持度：[高/中/低]
```

### Prompt 1.2 - 分层次润色（Light/Medium/Heavy）

**使用场景**：根据润色需求选择不同强度

```
我需要对论文进行分层次的润色。

【原文内容】
[粘贴你的论文]

【润色等级】
请按照以下等级进行润色：
[选择等级：Light / Medium / Heavy]

【各等级要求】

**Light 级别（轻度润色）**：
- 仅去除最明显的 AI 套话
- 微调句式，保持原文结构
- 不大幅改写
- 适用于：原文质量较好，只需微调

**Medium 级别（中度润色）**：
- 去除明显套话
- 调整句式，使其更自然
- 统一文风和术语
- 删除冗余表达
- 适用于：大部分 AI 生成内容

**Heavy 级别（深度润色）**：
- 深度改写，接近人类学者写作风格
- 完全去除模板化痕迹
- 重构段落逻辑
- 精简表达，提升可读性
- 适用于：原文 AI 痕迹明显，需要大幅调整

【输出要求】
1. 说明你实际执行的润色操作
2. 输出润色后的文本
3. 列出主要修改点
```

### Prompt 1.3 - 段落级去AI化（逐段润色）

**使用场景**：对特定段落进行精细润色

```
我需要对以下段落进行去AI化润色。

【段落内容】
[粘贴需要润色的段落]

【背景信息】
这段论文属于：
- 论文部分：[引言/方法/结果/讨论/结论]
- 所属学科：[计算机/医学/物理等]
- 写作风格偏好：[简练/详实/等等]

【润色要求】
1. 识别并改写 AI 套话
2. 转换为自然学术语气
3. 保持逻辑清晰
4. 提升可读性
5. 与上下文衔接自然

【输出格式】
【原文】
[原段落内容]

【问题识别】
- 发现的套话/问题：[列出]
- 需要改进的地方：[列出]

【润色版本1（温和版）】
[改动较小的版本]

【润色版本2（激进版）】
[改动较大的版本]

【改写说明】
解释每个版本的修改策略和适用场景
```

### Prompt 1.4 - AI模板检测（诊断工具）

**使用场景**：检测论文中可能存在的AI模板化痕迹

```
请帮我检测以下论文中是否存在AI模板化痕迹。

【论文内容】
[粘贴论文全文或长段落]

【检测任务】
请扫描全文，识别并标记：

1. **常见AI套话**：
   - 开场白："在当今时代"、"随着..."等
   - 过渡句："值得注意的是"、"另一方面"等
   - 结尾套话："综上所述"、"因此"等
   - 模板表达："研究表明"、"根据数据"等

2. **生硬表达**：
   - 过度使用被动语态
   - 长句过多且结构单一
   - 重复句式

3. **冗余表达**：
   - 语义重复
   - 空洞词汇（"某种程度"等）
   - 过度修饰

4. **不自然语气**：
   - AI 特有的句式结构
   - 不恰当的连接词
   - 缺乏学者常用的表达

【输出格式】
【检测报告】

【问题类型统计】
- 常见AI套话：发现 X 处
- 生硬表达：发现 X 处
- 冗余表达：发现 X 处
- 不自然语气：发现 X 处

【详细问题列表】

1. AI套话（按出现顺序）
   位置：[段落 X，第 Y 句]
   原文：[引用原文]
   问题：[说明为什么是AI套话]
   建议：[如何改写]

2. 生硬表达
   ...

3. 冗余表达
   ...

4. 不自然语气
   ...

【总体评估】
- AI 化程度：[低/中/高]
- 建议润色等级：[Light/Medium/Heavy]
- 优先处理位置：[列出最需要改进的段落]
```

### Prompt 1.5 - 文风一致性检查

**使用场景**：确保全文学术风格统一

```
请帮我检查以下论文的文风一致性。

【论文全文】
[粘贴完整论文]

【检查维度】
请从以下维度检查一致性：

1. **时态使用**
   - 引言部分应该用现在时描述研究现状
   - 方法部分应该用过去时描述已完成的操作
   - 结果和讨论根据需要混合时态
   - 检查全文时态是否正确、一致

2. **人称使用**
   - 是否一致使用第一人称复数（"我们"）
   - 或者一致使用被动语态（避免人称）
   - 检查是否存在混用

3. **术语统一**
   - 同一个概念是否使用相同术语
   - 是否存在不同章节用词不同的情况
   - 缩写词的定义和使用是否一致

4. **句式风格**
   - 句长分布是否合理（是否有段落全是非常长或非常短的句子）
   - 是否某一段落句式过于单一
   - 主动/被动语态比例是否合适

5. **引用格式**
   - 引用格式是否全文统一
   - 多个引用的排列是否一致
   - 文内引用格式是否统一

6. **表达风格**
   - 是否全文使用一致的学术语气
   - 是否某几段与其他段落风格差异明显

【输出格式】
【一致性检查报告】

【时态一致性】
- 发现的问题：[列出]
- 建议：[如何修正]

【人称一致性】
- 发现的问题：[列出]
- 建议：[如何修正]

【术语一致性】
- 发现的不一致术语：
  概念A：在X段用"术语1"，在Y段用"术语2"
  ...
- 建议：[统一为哪个术语]

【句式风格】
- 某段落句式问题：
  段落X：[问题描述]
  ...
- 建议：[如何改进]

【引用格式】
- 发现的不一致引用：
  引用1：格式A
  引用2：格式B
  ...
- 建议：[统一格式]

【表达风格】
- 发现的不一致表达：
  段落X：[问题描述]
  ...
- 建议：[如何调整]

【总体评分】
- 文风一致性评分：[1-10分]
- 主要问题：[总结核心问题]
- 改进建议：[优先处理哪些问题]
```

### Prompt 1.6 - 快速去AI化（5分钟版）

**使用场景**：快速去除最明显的AI痕迹

```
请快速帮我去除这段文字中最明显的AI痕迹。

【内容】
[粘贴文本]

【快速处理原则】
仅做以下处理（不要大幅改写）：
1. 删除"在当今时代"、"随着...的发展"等开场套话
2. 删除"值得注意的是"、"值得一提的是"等过渡套话
3. 删除重复的"重要"、"关键"等修饰词
4. 删除"某种程度"、"某种程度上"等模糊表达
5. 简单调整明显生硬的句式

【输出】
直接给出处理后的版本，不要解释细节。
```

## 🎯 使用建议

### 使用顺序推荐

1. **第一步**：用 Prompt 1.4（AI模板检测）诊断问题
2. **第二步**：根据诊断结果选择润色等级
3. **第三步**：用 Prompt 1.1（全文去AI化润色）或 Prompt 1.2（分层次润色）
4. **第四步**：用 Prompt 1.5（文风一致性检查）确保统一
5. **第五步**：人工审阅和最终调整

### 选择指南

- **全文润色** → Prompt 1.1
- **需要控制润色强度** → Prompt 1.2
- **精细润色特定段落** → Prompt 1.3
- **先诊断再润色** → Prompt 1.4 → 1.1
- **检查文风一致性** → Prompt 1.5
- **快速处理** → Prompt 1.6
