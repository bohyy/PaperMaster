## 🎨 色盲友好型配色方案

### 方案 1：色盲友好型配色（CB-Friendly）

**适用场景**：所有需要考虑色盲读者的学术图表

```python
import matplotlib.pyplot as plt

# 色盲友好型配色（Wong 配色方案）
COLORBLIND_COLORS = {
    'blue': '#0072B2',      # 明亮的蓝色
    'orange': '#D55E00',     # 橙色
    'green': '#009E73',      # 蓝绿色
    'pink': '#CC79A7',       # 粉红色
    'red': '#E69F00',        # 深橙色
    'purple': '#56B4E9',     # 浅紫色
    'brown': '#F0E442',      # 黄色
    'gray': '#999999'        # 灰色
}

# 推荐色盲友好调色板
CB_PALETTE = [
    '#0072B2',  # 1
    '#D55E00',  # 2
    '#009E73',  # 3
    '#F0E442',  # 4
    '#CC79A7',  # 5
    '#56B4E9',  # 6
]

# 使用示例
plt.bar(['A', 'B', 'C'], [10, 20, 15], color=CB_PALETTE[:3])
plt.show()
```

**R (ggplot2) 版本**：


## 🔧 配色工具函数

```python
def get_color_palette(n_colors: int, scheme: str = 'colorblind') -> list:
    """
    获取指定数量的颜色

    Args:
        n_colors: 需要的颜色数量
        scheme: 配色方案 ('colorblind', 'nature', 'science', 'okabeito')

    Returns:
        颜色列表
    """
    schemes = {
        'colorblind': CB_PALETTE,
        'nature': NATURE_PALETTE,
        'science': SCIENCE_PALETTE,
        'okabeito': OKABE_ITO_COLORS
    }

    colors = schemes.get(scheme, CB_PALETTE)

    # 如果颜色不够，循环使用
    return (colors * (n_colors // len(colors) + 1))[:n_colors]


def apply_colorblind_palette(plt_module):
    """
    将 Matplotlib 设置为色盲友好型配色
    """
    import matplotlib as mpl
    mpl.rcParams['axes.prop_cycle'] = mpl.cycler('color', CB_PALETTE)
    plt_module.style.use('seaborn-v0_8-whitegrid')
```

## 💡 使用建议

### ✅ 推荐做法

1. **优先使用色盲友好型配色**：CB-Friendly 或 Okabe-Ito
2. **根据期刊选择**：Nature 用 Nature 配色，Science 用 Science 配色
3. **颜色数量不超过 7 种**：太多颜色难以区分
4. **黑白打印测试**：打印成黑白检查是否仍可区分
5. **保持一致性**：同一篇文章中的图表使用相同的配色方案

### ⚠️ 避免做法

- ❌ 使用红绿对比（红绿色盲无法区分）
- ❌ 使用过于明亮的颜色（打印时效果差）
- ❌ 使用过多颜色（超过 8 种）
- ❌ 使用渐变色（除非热力图等特殊场景）
- ❌ 忽略黑白打印效果
