# 学术 API 多源检索系统

## 🎯 功能简介

PaperMaster 集成了 5 个完全免费的学术 API，实现**多源自动切换**，确保文献检索功能永远可用。

## ✨ 支持的学术 API

| API | 特点 | 优势 | 限额 |
|-----|------|------|------|
| **Semantic Scholar** | AI 驱动，数据质量高 | 推荐论文、引用分析、影响力评估 | 100次/分钟 |
| **OpenAlex** | 开放学术图谱，覆盖面广 | 全球最全的学术数据 | 基本无限制 |
| **CORE** | 开放获取论文库 | 优先返回 OA 论文，可下载全文 | 10万次/月 |
| **CrossRef** | DOI 解析权威 | 元数据最准确，DOI 查询 |基本无限制 |
| **arXiv** | 预印本库，更新最快 | 最新论文抢先看，物理/计算机/AI | 10万次/月 |

## 🔄 自动降级切换逻辑

### 工作原理

```
用户请求 → API管理器 → 尝试API 1
                         ↓
                      成功？→ 返回结果
                         ↓ 失败
                      尝试API 2
                         ↓
                      成功？→ 返回结果
                         ↓ 失败
                      ...
                         ↓
                      尝试API 5
                         ↓
                      成功？→ 返回结果
                         ↓ 失败
                      返回缓存结果 / 提示建议
```

### 切换触发条件

1. **API 请求失败**：网络错误、服务器宕机
2. **速率限制**：超过 API 调用频率限制
3. **额度耗尽**：月度配额用完
4. **超时**：响应时间超过 10 秒
5. **数据不完整**：返回数据缺少必要字段

### API 优先级（默认）

```
1. Semantic Scholar  → 数据质量最高，优先使用
2. OpenAlex         → 覆盖面最广，作为主力
3. CORE              → OA 论文优先，可下载全文
4. CrossRef          → DOI 查询权威
5. arXiv             → 预印本快速获取
```

## 💡 使用建议

### ✅ 最佳实践

1. **依赖自动切换**：不需要手动指定 API
2. **信任多源聚合**：多个 API 数据互补，更全面
3. **定期更新缓存**：确保获取最新数据
4. **检查 OA 状态**：优先使用开放获取论文
5. **验证 DOI**：DOI 是最稳定的论文标识

### ⚠️ 注意事项

- arXiv 仅适用于预印本，正式论文需要其他 API
- 不同 API 的数据质量有差异，优先使用 Semantic Scholar
- 大量查询时注意 API 速率限制
- 全文下载仅限开放获取论文

## 🎯 核心优势

✅ **永远可用**：5 个 API 互为备份，单一 API 故障不影响使用
✅ **自动切换**：用户无感知，自动选择最佳 API
✅ **数据互补**：不同 API 提供不同维度的数据，聚合后更全面
✅ **零成本**：所有 API 完全免费，无需 API Key
✅ **智能缓存**：提升性能，减少 API 调用
✅ **开放获取优先**：CORE 确保优先返回可下载全文的论文
