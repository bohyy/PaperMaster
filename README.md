
# 📝 PaperMaster - 你的专属学术写作全能助手
> 一站式搞定从选题、读文献、写论文、润色、画图到投稿的全流程科研需求，科研效率直接拉满✨
---
## 🌟 核心功能（89+模板全覆盖）
| 模块 | 功能说明 |
|------|----------|
| ✍️ **去AI化润色** | 6种顶会/顶刊风格适配（Nature/Science/IEEE/CCF A类等），3档润色力度，完全去掉AI生成痕迹，符合真人学术写作语气 |
| 🔍 **多源文献检索** | 5个免费学术API（Semantic Scholar/OpenAlex/CORE/CrossRef/arXiv）自动切换，查文献找引用永远不翻车 |
| 🎨 **学术绘图模板** | 色盲友好配色，覆盖顶会/顶刊规范，直接生成Python/R绘图代码，改数据就能用 |
| 📋 **全流程写作模板** | 从选题、文献综述、摘要、引言、方法论、结果讨论到结论，89+现成Prompt直接用 |
| 🧪 **实验辅助工具** | 实验设计方案、消融实验模板、显著性计算、SOTA结果自动对比，不用再查资料 |
| 📮 **投稿辅助工具** | Cover Letter生成、审稿人回复模板、选刊推荐、投稿时间线规划，投稿不赶死线 |
| 🔧 **通用工具** | 引用格式一键转换（IEEE/GB7714/APA等）、专业术语校验、降重改写、逻辑检查

## 🚀 快速使用
### 1. CLI 命令行工具（跨平台可用）
```bash
# 安装后直接用
papermaster refine --level medium 你的论文.md
papermaster refine --journal Nature 你的论文.md
papermaster figure use lineplot --language python
papermaster prompt search "润色"
```
### 2. OpenClaw Skill 直接对话
直接用自然语言说需求就行：
> 「帮我润色这段摘要，符合IEEE期刊风格，去掉AI味道」
> 「帮我选个机器学习方向的CCF A类会议，录用率高审稿快」
> 「帮我生成多模态大模型方向的消融实验设计方案」
> 「帮我写个回复审稿人的框架，意见是说我创新点不足」

## 📦 安装方法
### OpenClaw Skill 安装
```bash
# 直接放到OpenClaw技能目录即可
mv PaperMaster ~/.openclaw/skills/
```
### CLI 工具安装
```bash
# 放到系统路径
sudo cp cli/papermaster.py /usr/local/bin/papermaster
sudo chmod +x /usr/local/bin/papermaster
```
