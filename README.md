## 🌟 核心功能全覆盖（89+专业模板）

### ✍️ 1. 去AI化润色模块（核心亮点）
- **6种期刊风格适配**：支持Nature、Science、IEEE、Springer、CCF A类会议、国内毕业论文等多种写作风格，一键切换
- **3档润色力度**：
  - Light：仅修正语法错误和专业术语
  - Medium：调整句式逻辑，去除AI模板化表述
  - Heavy：深度改写，完全匹配真人学术写作语气，无任何AI痕迹
- **一致性检查**：自动统一专业术语、时态、格式，符合期刊规范
- **降重功能**：改写重复率高的段落，保持原意不变，符合学术规范
### 🔍 2. 多源文献检索模块
- **5个免费学术API自动切换**：Semantic Scholar、OpenAlex、CORE、CrossRef、arXiv，单一API故障自动降级，永远可用
- **全流程文献处理功能**：
  - 文献精准翻译：针对学术内容优化翻译，专业术语100%准确
  - 文献速读总结：自动提炼核心创新点、实验方法、关键结果、研究不足，1分钟读完一篇文献
  - 文献综述生成：输入多篇参考文献，自动梳理研究脉络、分类总结现有工作、指出研究gap
  - 引用格式一键转换：支持IEEE、GB7714、APA、MLA等所有主流格式，直接生成BibTeX
### 🎨 3. 学术绘图模板模块
- **专业配色方案**：内置色盲友好型配色（Wong、Okabe-Ito）、Nature/Science/IEEE等顶刊推荐配色
- **5类常见绘图模板**：折线图、柱状图、散点图、热力图、箱线图，覆盖90%学术绘图需求
- **双语言代码生成**：同时输出Python（Matplotlib/Seaborn）和R（ggplot2）代码，改数据就能直接用
- **自动标注规范**：自动生成符合期刊要求的坐标轴标签、图例、比例尺，不用手动调整
### 📋 4. 全流程写作模板模块
- **89+精心设计的Prompt模板**，覆盖写作全流程：
  - 选题调研：研究方向挖掘、创新点提炼、研究现状调研
  - 论文写作：摘要、引言、方法论、实验结果、讨论、结论、参考文献各个部分的写作模板
  - 逻辑优化：段落逻辑调整、论证结构优化、专业术语校验
### 🧪 5. 实验研究辅助模块
- **实验设计助手**：覆盖计算机/生物/社科等常见领域的标准实验设计、消融实验方案，自动检查对照组是否完整
- **结果分析工具**：输入实验数据自动计算显著性（t检验、方差分析）、生成对比表格、提炼核心结论
- **SOTA自动对比**：自动匹配你研究方向近1年顶会顶刊SOTA结果，生成对比表格
### 📮 6. 投稿辅助工具模块
- **选刊推荐**：内置完整CCF/中科院分区/JCR分区数据库，支持按研究方向、录用率、审稿周期、APC费用筛选最合适的期刊/会议
- **文档生成**：Cover Letter自动生成、审稿人回复模板、投稿时间线规划，自动预留返修缓冲时间
- **投稿检查清单**：自动检查格式、字数、参考文献格式、图表规范，避免低级错误
### 🔧 7. 通用效率工具模块
- 专业术语统一校验
- 论文逻辑结构检查
- 中英文专业术语互译
- 文献引用自动去重

## 🚀 快速使用
### 1. CLI 命令行工具（跨平台可用）
```bash
# 去AI化润色，适配Nature风格
papermaster refine --level medium --journal Nature 你的论文.md
# 生成折线图Python代码
papermaster figure use lineplot --language python
# 搜索相关学术prompt
papermaster prompt search "审稿人回复"
# 转换引用格式为IEEE
papermaster cite convert "你的引用内容" --format IEEE
```
### 2. OpenClaw Skill 直接对话（最方便）
不用记命令，直接用自然语言说需求就行：
> 「帮我润色这段摘要，符合IEEE期刊风格，去掉AI味道」
> 「帮我选个机器学习方向的CCF A类会议，录用率>20%，审稿周期<3个月」
> 「帮我生成多模态大模型方向的消融实验设计方案」
> 「帮我写个回复审稿人的框架，意见是说我创新点不足」

## 📦 安装方法
### OpenClaw Skill 安装
```bash
# 直接把PaperMaster文件夹放到OpenClaw技能目录即可
mv PaperMaster ~/.openclaw/skills/
```
### CLI 工具全局安装
```bash
# 复制到系统路径即可使用
sudo cp cli/papermaster.py /usr/local/bin/papermaster
sudo chmod +x /usr/local/bin/papermaster
```
